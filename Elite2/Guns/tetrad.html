Overview

The Tetrad QS-4 is a quad shot Nerf blaster that was released in 2022 under the Elite 2.0 series for 11 dollars.

Specifications

  • Feet Per Second: 68 FPS
  • Ammo Type: Standard Elite
  • Capacity: 4 Darts
  • Firing Mechanism: Springer with smart selection
  • Accessories: None

Features

The Tetrad QS-4 is a pump-action blaster with a vertical set of four dart barrels. It features the use of intelligent air restrictors to allow a single dart to be fired from any barrel. It also features an integrated shoulder stock. It has an iron sight atop its muzzle and a single tactical rail above the blaster.
